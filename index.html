In-Place Content Editing

This example shows how user interface elements in a form can be created dynamically and used to edit the contents of a regular HTML list construct. Items can be added, deleted, and modified on-the-fly. The code is cross-browser, and works in IE 5.5 and later, Firefox 1.0+, and Mozilla 6 and later.

Click on an item in the list to invoke the UI for editing and deleting items.
